Présentation

Système de gestion pour ludothèque

Ce projet consiste à développer une base de données complète pour une ludothèque, permettant de gérer efficacement le catalogue de jeux (ajout, modification, suivi des stocks), les adhérents (inscriptions, historique d'emprunts), ainsi que les prêts et réservations (gestion des dates, alertes de retard). Le système devra également générer des rapports statistiques sur les jeux les plus populaires et les fréquentations, tout en offrant une interface conviviale pour les gestionnaires et les usagers. La solution sera conçue via un MCD UML avant implémentation technique.

Preuves

1. Il met à jour de manière pertinente une base de données relationnelle ✓

2. Il effectue des interrogations efficaces et affiche bien les résultats ✓

3. Il conçoit la BD en analysant le cahier des charges ✓

4. Il assure la cohérence et la qualité de la base de données ✓

5. Il s’appuie sur les modèles de données utilisés ✓
